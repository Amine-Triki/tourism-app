# Egypt Tourism App 🌴

تطبيق سياحي متقدم يتيح للمستخدمين استكشاف أجمل الوجهات السياحية المصرية والأنشطة المتاحة بها.

**An advanced tourism app that allows users to explore the most beautiful Egyptian tourism destinations and available activities.**

## المميزات الرئيسية | Key Features ✨

- 🗺️ **استكشاف الوجهات** - قائمة شاملة بأفضل الوجهات السياحية المصرية (أسوان، داهب، الأقصر، الغردقة، شرم الشيخ، مرسى علم)
- 📸 **معرض الصور** - صور عالية الجودة مع تأثيرات انتقالية جميلة
- 🎯 **الأنشطة والفعاليات** - اكتشف الأنشطة السياحية المتاحة في كل وجهة
- 🎨 **واجهة مستخدم حديثة** - تصميم أنيق بألوان متناسقة
- 📱 **تطبيق متعدد المنصات** - يعمل على الهاتف المحمول والويب والتطبيقات سطح المكتب
- 🧭 **الملاحة السهلة** - Bottom Navigation Bar لسهولة التنقل بين الأقسام

## التكنولوجيا المستخدمة | Technologies

- **Flutter** - إطار عمل لتطوير التطبيقات متعددة المنصات
- **Dart** - لغة البرمجة الأساسية
- **Material Design** - نظام التصميم
- **Font Awesome Icons** - مكتبة الأيقونات

## البدء السريع | Getting Started

### المتطلبات | Requirements
- Flutter SDK (^3.5.0)
- Dart SDK

### التثبيت والتشغيل | Installation

```bash
# استنساخ المشروع
git clone <repository-url>

# الانتقال إلى مجلد المشروع
cd flutter-first

# تثبيت الحزم
flutter pub get

# تشغيل التطبيق
flutter run
```

## الحزم المستخدمة | Dependencies

- `flutter` - SDK الأساسي
- `cupertino_icons` - أيقونات iOS
- `font_awesome_flutter` - أيقونات Font Awesome
