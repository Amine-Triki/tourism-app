# Egypt Tourism App 🌴

**An advanced tourism app that allows users to explore the most beautiful Egyptian tourism destinations and available activities.**

تطبيق سياحي متقدم يتيح للمستخدمين استكشاف أجمل الوجهات السياحية المصرية والأنشطة المتاحة بها.

## Key Features ✨ | المميزات الرئيسية

- 🗺️ **Explore Destinations** - A comprehensive list of the best Egyptian tourism destinations (Aswan, Dahab, Luxor, Hurghada, Sharm El-Sheikh, Marsa Alam)
- 📸 **Photo Gallery** - High-quality photos with beautiful transition effects
- 🎯 **Activities and Events** - Discover available tourism activities in each destination
- 🎨 **Modern User Interface** - Elegant design with harmonious colors
- 📱 **Multi-Platform App** - Works on mobile, web, and desktop applications
- 🧭 **Easy Navigation** - Bottom Navigation Bar for easy navigation between sections

---Technologies | التكنولوجيا المستخدمة

- **Flutter** - Framework for developing multi-platform applications

- **Material Design** - Design system
